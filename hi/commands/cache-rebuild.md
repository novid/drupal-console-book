# cache:rebuild
सभी साइट caches को पुनर्निर्माण और साफ़ करें।

**Usage:**
```
drupal cache:rebuild [arguments]
cr
cc
```

## Available arguments
Argument | Details
---------|-------------
cache | केवल एक विशेष caches साफ़ करें।

## Examples
* सभी caches के पुनर्निर्माण 
```
Drupal cr all
```
* डिस्कवरी caches के पुनर्निर्माण
```
Drupal cr discovery
```
