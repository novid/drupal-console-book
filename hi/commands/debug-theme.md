# debug:theme
एप्लीकेशन के लिए वर्तमान थीम्स को प्रदर्शित करता है

**Usage:**
```
drupal debug:theme [arguments]
dt
tde
```

## Available arguments
Argument | Details
---------|-------------
theme | डिबग करने के लिए विशिष्ट थीम

## Examples
* List of themes on the site
```
drupal debug:theme
```
* Bartik theme information
```
drupal debug:theme bartik
```
