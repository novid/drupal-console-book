# debug:theme
Afișează temele curente pentru aplicație

**Usage:**
```
drupal debug:theme [arguments]
dt
tde
```

## Available arguments
Argument | Details
---------|-------------
theme | O anumită temă ce va fi depanată

## Examples
* List of themes on the site
```
drupal debug:theme
```
* Bartik theme information
```
drupal debug:theme bartik
```
