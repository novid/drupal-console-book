# image:styles:flush
按图像样式执行 flush 函数或对所有图像样式执行 flush 函数

**使用方法:**
```
drupal image:styles:flush [arguments]
isf
```

## 可用参数
参数 | 详细信息
---------|-------------
styles | 图像样式名称

## 例子
* Flush large image style
```
drupal image:styles:flush  large
```
* Flush thumbnail image style
```
drupal image:styles:flush  thumbnail
```
