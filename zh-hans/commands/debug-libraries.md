# debug:libraries
显示应用程序中可用的库

**使用方法:**
```
drupal debug:libraries [arguments]
dl
```

## 可用参数
参数 | 详细信息
---------|-------------
group | 输入库名称

## 例子
* 列出所有可用的库
```
drupal debug:libraries
```
* 列出区块库信息
```
drupal debug:libraries block
```
* List block/drupal.block library
```
drupal debug:libraries block/drupal.block
```
