# module:uninstall
卸载模块

**使用方法:**
```
drupal module:uninstall [arguments] [options]
mou
```

## 可用选项
选项 | 详细信息
-------|-------------
--force | 忽略依赖并强制卸载这个模块?
--composer | 使用 Composer卸载模块

## 可用参数
参数 | 详细信息
---------|-------------
module | 输入模块名称

## 例子
* Uninstall the module specifying the module name
```
drupal module:uninstall  modulename
```
