# user:login:url
返回一次性登录链接.

**使用方法:**
```
drupal user:login:url [arguments]
ulu
usli
uli
```

## 可用参数
参数 | 详细信息
---------|-------------
user | User name/id.

## 例子
* Get one time login url for user id 1
```
drupal user:login:url  1
```
* Get one time login url for username jmolivas
```
drupal user:login:url jmolivas
```
