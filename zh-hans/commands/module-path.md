# module:path
返回模块的相对路径(或绝对路径)

**使用方法:**
```
drupal module:path [arguments] [options]
mop
```

## 可用选项
选项 | 详细信息
-------|-------------
--absolute | 返回模块绝对路径

## 可用参数
参数 | 详细信息
---------|-------------
module | 模块名称 (机读名称)

## 例子
* Get the relative path of the module specifying the module name
```
drupal module:path  modulename
```
