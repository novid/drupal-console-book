# migrate:setup
导入或创建数据库迁移

**使用方法:**
```
drupal migrate:setup [options]
mis
```

## 可用选项
选项 | 详细信息
-------|-------------
--db-type | Drupal 数据库类型
--db-host | 数据库主机
--db-name | 数据库名称
--db-user | 数据库用户
--db-pass | 数据库密码
--db-prefix | 数据库前缀
--db-port | 数据库端口
--source-base_path | Local file directory containing your source site (e.g. /var/www/docroot), or your site address (for example http://example.com)
