# update:execute
执行的给出的某个模块的更新程序或者执行所有的更新程序

**使用方法:**
```
drupal update:execute [arguments]
upex
updb
```

## 可用参数
参数 | 详细信息
---------|-------------
module | 模块名
update-n | commands.update.execute.options.update-n

## 例子
* Update all entities
```
drupal update:execute
```
* Execute updates for system module
```
drupal update:execute system
```
