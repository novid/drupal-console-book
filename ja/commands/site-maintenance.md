# site:maintenance
サイトをメンテナンスモードに切り替える

**Usage:**
```
drupal site:maintenance [arguments]
sma
```

## Available arguments
Argument | Details
---------|-------------
mode | メンテナンスモード

## Examples
* Switch on maintenance
```
drupal site:maintenance  on
```
* Switch off maintenance
```
drupal site:maintenance  off
```
