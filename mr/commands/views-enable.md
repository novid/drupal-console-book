# views:enable
एक दृश्य सक्षम करा.

**वापर:**
```
drupal views:enable [arguments]
ve
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
view-id | दृश्य आयडी

## उदाहरणे
* सामग्री दृश्य सक्षम करा.
```
drupal views:enable  content
```
* FrontPage view सक्षम करा
```
drupal views:enable  frontpage
```
