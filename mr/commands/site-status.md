# site:status
वर्तमान Drupal स्थापना स्थिती पहा.

**वापर:**
```
drupal site:status [options]
ss
st
```

## उपलब्ध पर्याय
पर्याय | तपशील
-------|-------------
--format | commands.site.status.options.format

## उदाहरणे
* टेबल म्हणून आउटपुट स्वरूपन निर्दिष्ट करणारे Drupal स्थापना स्थिती मिळवा.
```
drupal site:status  \
  --format="table"
```
