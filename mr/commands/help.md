# help
आदेशासाठी मदत दर्शवितो.

**वापर:**
```
drupal help [arguments] [options]
```

## उपलब्ध पर्याय
पर्याय | तपशील
-------|-------------
--xml | क्सएमएलच्या रूपात आऊटपुट सूचीमध्ये.
--raw | कच्च्या आदेश सूचीचे आउटपुट करण्यासाठी.
--format | आउटपुट स्वरूप (txt, xml, json, किंवा md)

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
command_name | आदेशाचे नाव
