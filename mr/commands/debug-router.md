# debug:router
अनुप्रयोग चालू मार्ग दाखवतो

**वापर:**
```
drupal debug:router [arguments]
dr
rod
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
route-name | मार्ग नावे

## उदाहरणे
* अनुप्रयोगासाठी वर्तमान मार्ग प्रदर्शित करते.
```
drupal rod
```
* मार्ग user.page (/ वापरकर्ता) साठी तपशील प्रदर्शित करते.
```
drupal rod user.page
```
