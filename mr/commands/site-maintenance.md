# site:maintenance
साइटला देखभाल मोडमध्ये स्विच करा.

**वापर:**
```
drupal site:maintenance [arguments]
sma
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
mode | साइट देखभाल मोड.

## उदाहरणे
* देखरेखीवर स्विच करा.
```
drupal site:maintenance  on
```
* देखभाल बंद करा.
```
drupal site:maintenance  off
```
