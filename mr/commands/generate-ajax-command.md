# generate:ajax:command
Generate & Register a custom ajax command

**वापर:**
```
drupal generate:ajax:command [options]
gac
```

## उपलब्ध पर्याय
पर्याय | तपशील
-------|-------------
--module | मॉड्यूलचे नाव.
--class | Ajax Class name
--method | Custom ajax method
--js-name | Custom javascript name
