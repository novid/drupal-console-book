# test:run
ऍप्लिकेशनसाठी उपलब्ध असलेल्या चाचण्यांमधून टेस्ट युनिट चालवा.

**वापर:**
```
drupal test:run [arguments] [options]
ter
```

## उपलब्ध पर्याय
पर्याय | तपशील
-------|-------------
--url | Test url

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
test-class | चाचणी वर्ग.
test-methods | धावण्याची कसोटी पद्धत.
