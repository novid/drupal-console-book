# user:login:url
एक-प्रयोगकर्ता लॉगिन url मिळवते.

**वापर:**
```
drupal user:login:url [arguments]
ulu
usli
uli
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
user | User name/id.

## उदाहरणे
* यूजर आयडी 1 ला एकदा लॉगिन यूआरएल मिळवा.
```
drupal user:login:url  1
```
* Get one time login url for username jmolivas
```
drupal user:login:url jmolivas
```
