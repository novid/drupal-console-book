# debug:theme
अनुप्रयोगासाठी वर्तमान थीम प्रदर्शित करते.

**वापर:**
```
drupal debug:theme [arguments]
dt
tde
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
theme | डीबग करण्यासाठी विशिष्ट थीम.

## उदाहरणे
* साइटवरील थीमची सूची.
```
drupal debug:theme
```
* बार्टिक थीमची माहिती.
```
drupal debug:theme bartik
```
