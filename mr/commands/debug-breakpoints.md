# debug:breakpoints
अनुप्रयोग मध्ये उपलब्ध असलेले ब्रेकपॉइंट प्रदर्शित करते.

**वापर:**
```
drupal debug:breakpoints [arguments]
dbre
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
group | ब्रेकपॉईंट गट नाव प्रविष्ट करा.

## उदाहरणे
* समूह नाव प्रदान करा.
```
drupal breakpoints:debug bartik
```
