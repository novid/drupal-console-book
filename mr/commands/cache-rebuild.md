# cache:rebuild
सर्व साईट caches स्वच्छ तसेच पुन्हानिर्मित.

**वापर:**
```
drupal cache:rebuild [arguments]
cr
cc
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
cache | फक्त ठराविक cache साफ.

## उदाहरणे
* सर्व caches पुन्हानिर्मिती करा.
```
drupal cr all
```
* discovery cache पुन्हा निर्मिती करा.
```
drupal cr discovery
```
