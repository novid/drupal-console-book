# cron:execute
क्रॉन लागू करणारे काही विशेष मोड्यूल चालावा अथवा सर्व क्रॉन चालवा.

**वापर:**
```
drupal cron:execute [arguments]
croe
cre
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
module | मॉड्यूलचे नाव.

## उदाहरणे
* जागतिक पातळीवर क्रोन चालवा.
```
drupal cron:execute
```
* निर्दिष्ट केलेल्या मॉड्यूलवर क्रोन कार्यान्वित करा.
```
drupal cron:execute \
  <module>
```
