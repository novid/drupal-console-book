# site:mode
सिस्टम कार्यक्षमता कॉन्फिगरेशन स्विच करा.

**वापर:**
```
drupal site:mode [arguments]
smo
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
environment | पर्यावरण नाव[dev, prod]

## उदाहरणे
* prod करण्यास प्रणाली स्विच करा.
```
drupal site:mode  prod
```
* dev करण्यास प्रणाली स्विच करा.
```
drupal site:mode  dev
```
