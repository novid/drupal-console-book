# debug:state
वर्तमान अवस्था की दाखवा.

**वापर:**
```
drupal debug:state [arguments]
dst
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
key | डीबग करण्यासाठी अवस्था की.

## उदाहरणे
* साइटवरील अवस्थांची यादी.
```
drupal debug:state
```
* अवस्थांच्या यादीतून install_task अवस्था तपशील दाखवतो.
```
drupal debug:state install_task
```
