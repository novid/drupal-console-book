# debug:rest
चालू रेस्ट अर्ज संसाधन प्रदर्शित करा

**वापर:**
```
drupal debug:rest [arguments] [options]
rede
```

## उपलब्ध पर्याय
पर्याय | तपशील
-------|-------------
--authorization | रेस्ट संसाधन स्टेट्स सक्षम | अक्षम

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
resource-id | रेस्ट आयडी

## उदाहरणे
* rest हुक प्रदर्शित करते.
```
drupal debug:rest
```
