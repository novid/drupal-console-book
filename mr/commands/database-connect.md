# database:connect
डीबी कनेक्शन दर्शविते.

**वापर:**
```
drupal database:connect [arguments]
dbco
sqlc
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
database | Settings.php पासून डेटाबेस की.

## उदाहरणे
* निर्दिष्ट केलेल्या डेटाबेसशी कनेक्ट होते किंवा डीफॉल्ट नसल्यास वितर्क पास केले जातात.
```
drupal database:connect \
  <database>
```
