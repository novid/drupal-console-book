# debug:event
वर्तमान इव्हेंट प्रदर्शित करा.

**वापर:**
```
drupal debug:event [arguments]
dev
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
event | डीबग करण्यासाठी इव्हेंट.

## उदाहरणे
* डीबग केलेल्या सर्व इव्हेंटची सूची करा.
```
drupal debug:evet
```
* kernel.request इव्हेंटसाठी माहिती दर्शवा.
```
drupal debug:event kernel.request
```
