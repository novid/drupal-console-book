# cache:tag:invalidate
Invalidate cache tags.

**वापर:**
```
drupal cache:tag:invalidate [arguments]
cti
```

## उपलब्ध वितर्क
वितर्क | तपशील
---------|-------------
tag | One or more tags to invalidate.

## उदाहरणे
* Invalidate routes
```
drupal cti routes
```
* Invalidate a specific node
```
drupal cti node:1 node_list
```
