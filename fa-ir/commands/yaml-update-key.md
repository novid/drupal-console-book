# yaml:update:key
Replace a YAML key in a YAML file.

**Usage:**
```
$ drupal yaml:update:key [arguments]
$ yu  
```

## Available arguments
Argument | Details
---------|-------------
yaml-file | Path of YAML file to update
yaml-key | YAML key to update
yaml-new-key | commands.yaml.update.value.arguments.yaml-new-key
