# user:password:hash
Generate a hash from a plaintext password.

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | Password(s) in text format

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
