# debug:views
Displays current views resources for the application

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | View tag
--status | View status (Enabled|Disabled)

## Available arguments
Argument | Details
---------|-------------
view-id | View ID

## Examples
* List of views on the site
```
drupal debug:views
```
