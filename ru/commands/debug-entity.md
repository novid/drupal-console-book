# debug:entity
Отладка сущностей, имеющихся в системе

**Использование:**
```
drupal debug:entity [arguments]
de
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
entity-type | commands.debug.entity.arguments.entity-type

## Примеры
* Вывести имеющиеся сущности
```
drupal debug:entity
```
