# state:delete
Удаляет состояние

**Использование:**
```
drupal state:delete [arguments]
std
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
name | Имя состояния.

## Примеры
* Удаляет состояние с указанным именем
```
drupal state:delete  comment.maintain_entity_statistics
```
