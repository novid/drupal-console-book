# debug:permission
Показывает все права пользователей.

**Использование:**
```
drupal debug:permission [arguments]
dp
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
role | Пользовательская роль

## Примеры
* Показывает все права, доступные на сайте
```
drupal debug:permission
```
