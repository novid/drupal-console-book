# generate:module:file
Сгенерировать файл .module

**Использование:**
```
drupal generate:module:file [options]
gmf
```

## Доступные параметры
Команда | Детали
-------|-------------
--module | Имя модуля.

## Примеры
* Сгенерировать файл .module, указав имя модуля
```
drupal generate:module:file  \
  --module="modulename"
```
