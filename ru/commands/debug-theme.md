# debug:theme
Отображает текущие темы для приложения

**Использование:**
```
drupal debug:theme [arguments]
dt
tde
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
theme | Тема для отладки

## Примеры
* Выводит темы сайта
```
drupal debug:theme
```
* Информация о теме Bartik
```
drupal debug:theme bartik
```
