# debug:router
Отображает текущие маршруты для приложения

**Использование:**
```
drupal debug:router [arguments]
dr
rod
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
route-name | Имена маршрутов

## Примеры
* Отображает текущие маршруты для приложения
```
drupal rod
```
* Отображает детали для маршрута user.page (/user)
```
drupal rod user.page
```
