# database:drop
Удалить все таблицы в данной базе данных.

**Использование:**
```
drupal database:drop [arguments]
dbd
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
database | Ключ баы данных из settings.php

## Примеры
* Удаляет таблицы базы данных переданной в аргументе
```
drupal database:drop \
  <database>
```
