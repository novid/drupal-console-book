# debug:migrate
Отображение текущей миграции, доступной для приложения

**Использование:**
```
drupal debug:migrate [arguments]
mid
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
tag | Тег миграции

## Примеры
* Отображает информацию о текущей миграции
```
drupal debug:migrate
```
