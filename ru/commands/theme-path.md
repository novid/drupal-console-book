# theme:path
Относительный путь к теме (или абсолютный)

**Использование:**
```
drupal theme:path [arguments] [options]
thp
```

## Доступные параметры
Команда | Детали
-------|-------------
--absolute | Абсолютный путь к теме

## Доступные аргументы
Аргумент | Детали
---------|-------------
theme | Имя темы

## Примеры
* Путь к теме mytheme
```
drupal theme:path  mytheme
```
