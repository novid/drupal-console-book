# theme:uninstall
Деинсталяция тем в прилодении

**Использование:**
```
drupal theme:uninstall [arguments]
thu
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
theme | список тем должен быть разделен запятыми

## Примеры
* Деинсталяция темы с именем mytheme
```
drupal theme:uninstall  mytheme
```
