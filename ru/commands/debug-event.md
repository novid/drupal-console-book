# debug:event
Показать текущие события

**Использование:**
```
drupal debug:event [arguments]
dev
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
event | Событие для отладки

## Примеры
* Вывести все события, доступные для отладки
```
drupal debug:event
```
* Показать информацию о событии kernel.request
```
drupal debug:event kernel.request
```
