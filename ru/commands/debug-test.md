# debug:test
Отображает список доступных юнит тестов для приложения.

**Использование:**
```
drupal debug:test [arguments] [options]
td
```

## Доступные параметры
Команда | Детали
-------|-------------
--test-class | Класс теста

## Доступные аргументы
Аргумент | Детали
---------|-------------
group | Группа

## Примеры
* Отображает список доступных юнит тестов
```
drupal debug:test
```
