# features:import
Импортирует конфигурацию модуля.

**Использование:**
```
drupal features:import [arguments] [options]
fei
```

## Доступные параметры
Команда | Детали
-------|-------------
--bundle | Имя бандла

## Доступные аргументы
Аргумент | Детали
---------|-------------
packages | Имя пакета
