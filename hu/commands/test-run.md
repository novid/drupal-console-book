# test:run
Egységteszt futtatása az alkalmazás számára elérhető tesztek közül

**Usage:**
```
drupal test:run [arguments] [options]
ter
```

## Available options
Option | Details
-------|-------------
--url | Test url

## Available arguments
Argument | Details
---------|-------------
test-class | Tesztosztály
test-methods | Test method(s) to be run
