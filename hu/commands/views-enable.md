# views:enable
Nézet engedélyezése

**Usage:**
```
drupal views:enable [arguments]
ve
```

## Available arguments
Argument | Details
---------|-------------
view-id | Nézet azonosítója

## Examples
* Enable content view
```
drupal views:enable  content
```
* Enable frontpage view
```
drupal views:enable  frontpage
```
