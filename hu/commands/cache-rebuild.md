# cache:rebuild
A webhely összes gyorsítótárának újraépítése és törlése.

**Usage:**
```
drupal cache:rebuild [arguments]
cr
cc
```

## Available arguments
Argument | Details
---------|-------------
cache | Csak egy adott gyorsítótár törlése.

## Examples
* Összes gyorsítótár újraépítése
```
drupal cr all
```
* Felderítési gyorsítótár újraépítése
```
drupal cr discovery
```
