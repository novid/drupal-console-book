# site:status
Veure l'estat actual de l'instal·lació de Drupal

**Ús:**
```
drupal site:status [options]
ss
st
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--format | commands.site.status.options.format

## Exemples
* Get drupal installation status specifying the output format as table
```
drupal site:status  \
  --format="table"
```
