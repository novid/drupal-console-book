# debug:test
Llistar els Test Units disponibles per l'aplicació.

**Ús:**
```
drupal debug:test [arguments] [options]
td
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--test-class | Classe de verificació

## Arguments disponibles
Argument | Detalls
---------|-------------
group | Grup

## Exemples
* 
```
drupal debug:test
```
