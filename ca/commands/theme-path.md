# theme:path
Returns the relative path to the theme (or absolute path)

**Ús:**
```
drupal theme:path [arguments] [options]
thp
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--absolute | Return theme absolute path

## Arguments disponibles
Argument | Detalls
---------|-------------
theme | Theme name

## Exemples
* Get the path of mytheme
```
drupal theme:path  mytheme
```
