# create:roles
Create dummy roles for your Drupal 8 application.

**Ús:**
```
drupal create:roles [options]
crr
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--limit | How many roles would you like to create

## Exemples
* Provide roles.
```
drupal create:roles
```
* Provide the number of roles to create
```
drupal create:roles
```
