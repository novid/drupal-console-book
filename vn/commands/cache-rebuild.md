# cache:rebuild
Xây dựng lại và làm sạch tất cả cache trên trang.

**Usage:**
```
drupal cache:rebuild [arguments]
cr
cc
```

## Available arguments
Argument | Details
---------|-------------
cache | Chỉ xóa một cache được chỉ định

## Examples
* Xây dựng lại all caches
```
drupal cr all
```
* Xây dựng lại discovery cache
```
drupal cr discovery
```
