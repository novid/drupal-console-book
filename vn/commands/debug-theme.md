# debug:theme
Hiển thị các themes hiện tại cho ứng dụng

**Usage:**
```
drupal debug:theme [arguments]
dt
tde
```

## Available arguments
Argument | Details
---------|-------------
theme | Theme nhất định để chỉnh lỗi

## Examples
* List of themes on the site
```
drupal debug:theme
```
* Bartik theme information
```
drupal debug:theme bartik
```
