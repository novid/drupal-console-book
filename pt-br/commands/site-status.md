# site:status
Exibir informações da atual instalação do Drupal

**Utilização:**
```
drupal site:status [options]
ss
st
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--format | commands.site.status.options.format

## Exemplos
* Get drupal installation status specifying the output format as table
```
drupal site:status  \
  --format="table"
```
