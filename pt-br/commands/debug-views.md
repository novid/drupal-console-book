# debug:views
Exibir informações sobre a View atual

**Utilização:**
```
drupal debug:views [arguments] [options]
vde
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--tag | Tag da View
--status | Status da View (Habilitado|Desabilitado)

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
view-id | ID da View

## Exemplos
* List of views on the site
```
drupal debug:views
```
