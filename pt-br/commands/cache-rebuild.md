# cache:rebuild
Reconstruir e limpar todos os caches do site.

**Utilização:**
```
drupal cache:rebuild [arguments]
cr
cc
```

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
cache | Apenas limpar um cache específico.

## Exemplos
* Reconstruir todos os caches
```
drupal cr all
```
* Reconstruir cache de discovery
```
drupal cr discovery
```
