# debug:test
Listar testes unitários disponíveis para a aplicação.

**Utilização:**
```
drupal debug:test [arguments] [options]
td
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--test-class | Classe do teste

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
group | Grupo

## Exemplos
* 
```
drupal debug:test
```
