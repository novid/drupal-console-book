# debug:views:plugins
Mostrar os plugins de views atuais para a aplicação

**Utilização:**
```
drupal debug:views:plugins [arguments]
dvp
```

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
type | Filtrar plugins de views por tipo

## Exemplos
* List of views plugins
```
drupal debug:views:plugins
```
