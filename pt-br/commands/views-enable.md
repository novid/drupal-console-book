# views:enable
Habilitar uma View

**Utilização:**
```
drupal views:enable [arguments]
ve
```

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
view-id | ID da View

## Exemplos
* Enable content view
```
drupal views:enable  content
```
* Enable frontpage view
```
drupal views:enable  frontpage
```
