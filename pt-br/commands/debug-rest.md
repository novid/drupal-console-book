# debug:rest
Exibir o recurso REST atual para a aplicação

**Utilização:**
```
drupal debug:rest [arguments] [options]
rede
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--authorization | Recurso REST habilitado | desabilitado

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
resource-id | REST ID

## Exemplos
* Displays rest hooks
```
drupal debug:rest
```
