# user:login:url
Returns a one-time user login url.

**Usage:**
```
drupal user:login:url [arguments]
ulu
usli
uli
```

## Available arguments
Argument | Details
---------|-------------
user | User name/id.

## Examples
* Get one time login url for user id 10
```
drupal user:login:url  10
```
* Get one time login url for username jmolivas
```
drupal user:login:url jmolivas
```
